# FCC Manager (FiveM Citizen Changer)

FiveMの `citizen` フォルダを「一瞬で」変更・管理するためのツールです。
大容量のパックでも、シンボリックリンク（Junction）を利用することでコピーの待ち時間をなくし、一瞬で適用できるように設計されています。

## 特徴
- **一瞬での切り替え**: ギガバイト単位のパックでもコピーを行わないため、待ち時間がゼロです。
- **バニラ復元機能**: 現在の標準ファイルを安全にバックアップし、いつでもデフォルトのバニラ状態に復元可能です。
- **UIテーマ**: Ocean, Crimson, Hacker, Midnight, Sunlightの多彩なテーマから好みの外観を選択可能。
- **安全性と透明性**: ソースコードは完全にオープン化（公開）されています。また、配布されているインストーラー(.exe)は、GitHubのサーバー上で公開されているコードから直接・自動でビルド(GitHub Actions)されているため、制作者のパソコンで不審なプログラムが混入される心配がないことが証明されています。

## 使い方（一般ユーザー向け）
1. [Releases](../../releases) ページから最新のインストーラー (`FCC Setup X.X.X.exe`) をダウンロードして実行します。
2. アプリが起動したら「参照」ボタンから、ご自身の `FiveM Application Data` フォルダを選択します。
3. `FiveM Application Data/FCC_Citizens` というフォルダが作成されるので、その中に任意の名前のフォルダを作り、カスタムの citizen データを配置してください。
4. FCC側の「更新」ボタンを押すとパックとして認識され、ワンクリックで適用できるようになります。

## 開発・ビルド方法（エンジニア向け）

```bash
# クローン
git clone <リポジトリのURL>
cd FCC-Manager

# 依存パッケージのインストール
npm install

# 開発モードの起動
npm run electron:dev

# インストーラーのビルド（dist-electronフォルダに出力されます）
npm run electron:build
```
